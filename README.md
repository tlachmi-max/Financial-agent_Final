# 💰 מתכנן פיננסי - גרסה 2.0 (Goal-Based)

אפליקציית תכנון פיננסי מבוססת יעדים עם פרופיל משפחתי, ניתוח השקעות והמלצות אישיות.

## 🎯 מה חדש בגרסה 2.0?

### ✨ תכונות חדשות:
- **👤 פרופיל משפחתי** - מצב משפחתי, גילאים, ילדים
- **🎯 ניהול יעדים** - פנסיה, הון עצמי, יעדי חיים
- **📊 ניתוח התקדמות** - מעקב אחר התקדמות ביעדים
- **💡 המלצות אישיות** - כמה להפקיד, איך להתאים סיכון
- **📈 ויזואליזציה משופרת** - דשבורד מרכזי עם גרפים

### 🔄 שיפורים קיימים:
- חישוב מס מדויק (פטור מזכה 67%)
- קצבה נומינלית וריאלית
- סימולטור פנסיה
- מפת דרכים עם משיכות
- דוחות מסכמים

## 📦 התקנה ופריסה

### דרישות מקדימות:
- שרת web (Apache/Nginx) או GitHub Pages
- דפדפן מודרני (Chrome/Firefox/Safari/Edge)

### שלב 1: הורדה
```bash
git clone <repository-url>
cd financial-planner-v2
```

### שלב 2: פריסה

#### אופציה A: פריסה מקומית (בדיקה)
```bash
# Python
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

פתח דפדפן: `http://localhost:8000`

#### אופציה B: GitHub Pages
1. העלה לrepository ב-GitHub
2. הגדרות → Pages → Source: `main` branch
3. שמור → קבל URL: `https://username.github.io/financial-planner-v2`

#### אופציה C: Netlify/Vercel
1. גרור את התיקייה לממשק
2. פרוס
3. קבל URL אוטומטי

## 💾 ניהול נתונים

### שמירה
- **אוטומטי:** כל שינוי נשמר מיד ב-localStorage
- **ידני:** כפתור "שמור" בכל טאב
- **ייצוא:** Excel / PDF דרך כפתורים בהדר

### גיבוי
```javascript
// פתח Console בדפדפן (F12)
// הדבק קוד זה כדי לייצא:
const backup = localStorage.getItem('financial_planner_data');
console.log(backup);
// העתק את התוצאה ושמור בקובץ .txt
```

### שחזור
```javascript
// פתח Console בדפדפן (F12)
// הדבק קוד זה עם הנתונים שגיבית:
const data = `YOUR_BACKUP_DATA_HERE`;
localStorage.setItem('financial_planner_data', data);
location.reload();
```

## 🔧 הגדרות מתקדמות

### שינוי ברירות מחדל

#### תשואות
קובץ: `script.js` → חפש `TAX_RATES`
```javascript
const TAX_RATES = {
    'מניות סחיר': 25,
    'תיק עצמאי': 25,
    'קרן נאמנות': 25,
    'קופת גמל': 25,
    'קרן השתלמות': 25, // 6+ שנים: 0%
    'פנסיה': 0,         // מס מחושב בנפרד
    'עו"ש': 0,
    'פיקדון': 15
};
```

#### אינפלציה
קובץ: `script.js` → חפש `INFLATION_RATE`
```javascript
const INFLATION_RATE = 0.02; // 2% שנתי
```

#### אופק ברירת מחדל
קובץ: `index.html` → חפש `value="20"`
```html
<input type="number" id="sumYears" value="20" min="1" max="50">
```

## 🐛 פתרון בעיות

### הנתונים נעלמו
- **סיבה:** ניקוי cache/cookies
- **פתרון:** ייבא גיבוי או התחל מחדש

### האפליקציה לא נטענת
- **סיבה:** קובץ חסר או שגוי
- **פתרון:** וודא שכל 5 הקבצים בתיקייה

### חישובים לא מדויקים
- **סיבה:** שדה ריק או לא תקין
- **פתרון:** וודא שכל השדות מלאים

### PWA לא עובד
- **סיבה:** לא על HTTPS
- **פתרון:** השתמש ב-GitHub Pages או Netlify
